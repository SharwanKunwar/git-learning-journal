# 🚀 Beginner Open Source Contribution Guide

A simple step-by-step guide to making your first contribution on GitHub.

---

# 📌 Workflow

```text
Fork → Clone → Branch → Edit → Commit → Push → Pull Request
```

---

# 1️⃣ Fork the Repository

Click the **Fork** button on the GitHub repository page.

This creates your own copy of the project.

---

# 2️⃣ Clone the Repository

```bash
git clone YOUR_REPOSITORY_URL
```

### Example

```bash
git clone https://github.com/your-username/project-name.git
```

Move into the project folder:

```bash
cd project-name
```

---

# 3️⃣ Create a New Branch

```bash
git switch -c your-branch-name
```

### Example

```bash
git switch -c sandesh
```

---

# 4️⃣ Make Changes

Edit files, add your contribution, then save the changes.

Check modified files:

```bash
git status
```

---

# 5️⃣ Add & Commit Changes

```bash
git add .
```

```bash
git commit -m "Added my contribution"
```

---

# 6️⃣ Push Changes

```bash
git push -u origin your-branch-name
```

---

# 7️⃣ Create a Pull Request

1. Open your forked repository on GitHub  
2. Click **Compare & Pull Request**  
3. Submit the Pull Request  

🎉 Congratulations! You made your first open source contribution.

---

# 💡 Helpful Commands

| Command | Purpose |
|---|---|
| `git status` | Check changes |
| `git add .` | Add files |
| `git commit -m "msg"` | Save changes |
| `git push` | Upload changes |
| `git pull` | Get latest updates |
