## 📘 What Did I Learn Today?
- 🌿 How to create a new branch in a repository
- 🔄 How to switch between branches

---

## 🛠️ Steps Followed

### 1️⃣ Locate Yourself in the Repository Folder
Make sure you are inside the repository folder and currently on the `main` branch.

---

### 2️⃣ Create a New Branch
```bash
git branch "new-branch-name"
```

---

### 3️⃣ View All Branches
```bash
git branch
```

This command shows the list of branches, where you can see the newly created branch. 👀

---

### 4️⃣ Switch to the New Branch
```bash
git switch "new-branch-name"
```
